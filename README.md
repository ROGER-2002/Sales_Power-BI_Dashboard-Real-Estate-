# Sales_Power-BI_Dashboard-Real-Estate-
This is a Real Estate Sales Power BI Project
Real Estate Analytics Dashboard (Sales Performance)

This project presents an interactive Real Estate Analytics Dashboard built using Microsoft Power BI Desktop, focusing on both market analysis and sales performance in the real estate domain.

Project Overview

The dashboard provides comprehensive insights into real estate data, enabling analysis of property trends, sales performance, and key business metrics across multiple cities and developers.

Key Features
Market Analysis
Analyzed property price trends across different cities and localities
Identified high-demand areas based on locality-level insights
Evaluated developer distribution and market presence

Sales Performance
Tracked key KPIs such as:
Total Revenue
Units Sold
Average Selling Price
Price per Sqft
Identified top-performing projects and localities
Analyzed sales trends over time and developer performance

Dashboard Insights
Revenue comparison across cities, developers, and projects
Demand analysis based on BHK configuration
Sales contribution by channel (Broker, Direct, Online)
Identification of top-performing real estate segments

Tools & Technologies
Microsoft Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling & Visualization

Files Included
.pbix file – Power BI Dashboard
.pdf file – Dashboard export for quick view
Dataset – Synthetic real estate sales dataset

Objective
The goal of this project is to demonstrate data analysis and visualization skills by transforming raw real estate data into meaningful insights that support data-driven decision-making.
